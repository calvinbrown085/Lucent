FAQ

Lucent can't find my HDHomeRun

Lucent discovers HDHomeRun tuners on your local network automatically. Make sure both the tuner and your Apple device are on the same Wi-Fi network (or LAN). If discovery still fails, you can enter the tuner's IP address manually in Settings. Some networks isolate wireless clients from each other ("AP isolation" or "client isolation") — turn that off on your router.

The guide is empty after I switched sources

Listings are joined to channels by an identifier that differs between guide sources. When you switch between Gracenote and XMLTV, Lucent rebuilds the mapping on the next guide refresh. If listings still don't appear, open Settings and check the per-channel XMLTV ID override — it should match what your guide source uses for that channel.

Streams stall or buffer constantly

Live HDHomeRun streams are sensitive to network jitter. Try moving the tuner to a wired connection if possible. If you're on Wi-Fi, prefer 5 GHz and stay within line-of-sight of your access point. If buffering persists, file an issue with your network setup and the channel that's stalling.

Channel up / down is slow

Lucent prewarms adjacent channels in the background, but it can only prewarm as many channels as you have free tuners. An HDHomeRun with 2 tuners can prewarm 1 channel at a time (one tuner is always reserved for the channel you're currently watching). Tuners with more channels prewarm more.

Gracenote shows the wrong listings

Gracenote keys listings by postal code and broadcast region. If your postal code spans two regions or sits near a boundary, you may see listings from the wrong side. Switch to an XMLTV source if accuracy matters.

Where's DVR? Series passes? Remote streaming?

Not in v1. Lucent v1 focuses on doing live TV well. Recording, remote access, and series management are on the roadmap but won't ship in v1.

Does Lucent work outside the US?

The XMLTV path works anywhere — point Lucent at any compliant XMLTV feed. Gracenote's postal-code lookup is US-only.

System requirements

  • Apple TV with tvOS 26.2 or later, or iPhone/iPad on iOS 18+
  • HDHomeRun tuner on the same Wi-Fi/LAN
  • An antenna or compatible HDHomeRun input
  • US postal code for Gracenote, or any XMLTV feed for elsewhere
